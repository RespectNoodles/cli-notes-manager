## cli-notes-manager
The goal is to build a maintainable and extensible terminal knowledge base system with strong navigation logic and minimal friction when adding new notes.

notes-cli runnable command

numbered menus + keys (B/M/S/G/U/C/H/Q)

Markdown rendering (Rich)

search across all .md

Git status detection + “Quick Sync” that prints commands (exec is opt-in via allow_git_exec)

tests + GitHub Actions CI
